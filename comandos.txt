-> Navegando no cmd

-> inincializa um repositorio git (Utilizando apenas uma vez)
git init

-> Verficar o status do repositorio
git status

-> Adicionar os novos arquivos no git para versionar
git add .

-> Salvar uma versão dos arquivos
git commit -m "Porque estou salvando???"

-> Git push envia informações do computador para a nuvem
git push

-> Baixe a atualização dos arquivos do Github para uma pasta jpá existente
git pull
